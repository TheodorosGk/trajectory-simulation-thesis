# Simulation of Multiple-Aspect Trajectories with Machine Learning Methods

This repository accompanies my diploma thesis on synthetic trajectory generation using machine learning methods.

The practical implementation is based on the original **TS-TrajGen** framework:

- Original repository: https://github.com/WenMellors/TS-TrajGen
- Original method: **TS-TrajGen: Continuous Trajectory Generation Based on Two-Stage GAN**

## Purpose of this Repository

This repository was created to document the code, workflow, and experimental setup used in my thesis.

The implementation is primarily based on the original TS-TrajGen codebase, with only minor modifications made in order to run the code successfully in my experimental environment.

## My Contribution

My contribution in this thesis mainly includes:

- studying and analyzing the TS-TrajGen framework
- setting up the environment and dependencies
- preparing and organizing the required data and execution steps
- making minor code modifications where necessary for successful execution
- running the training / generation pipeline
- documenting the workflow used in the thesis
